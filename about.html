About NoodleMart 🍜

NoodleMart is a modern online noodle shopping platform built using HTML, CSS and JavaScript. Our goal is to make noodle shopping fast, simple, and enjoyable.

🚀 Our Idea

To create a user-friendly e-commerce website where customers can easily browse, search and buy their favorite noodle brands.

⚙️ Technology

Built using pure HTML, CSS and JavaScript focusing on performance, animations and clean UI design.

🛒 Smart Cart

Dynamic cart system with quantity control, price calculation and smooth user experience.

🎓 Academic Project

Designed as a college project to demonstrate front-end development and e-commerce concepts.

Our Mission

To deliver a professional e-commerce experience using simple web technologies while maintaining clean design, animations and usability.
